# Inbox AI (Local AI)

## Description
Agent IA qui lit les emails et les résume automatiquement.

## Tech
- Python
- Ollama (LLM local)

## Installation

1. Installer Ollama
2. Lancer :
   ollama run llama3

3. Installer dépendances :
   pip install -r requirements.txt

4. Lancer :
   python email_reader.py
